# Dog Park Near Me

## Configuration and Installation
Local Setup
```
git clone https://github.com/rebeccahongsf/gatsby-dogpark.git
cd gatsby-dogpark
npm install # Install node_modules dependencies
```

Build Command(s)
```
gatsby develop # Starts development server at http://localhost:8000/
```
